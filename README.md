## 📌 Sobre o projeto
Este projeto é um **sorteador de números aleatórios**, onde o usuário define:  
- Quantidade de números a serem sorteados;  
- Valor mínimo (De);  
- Valor máximo (Até).  

O programa então sorteia os números sem repetições e exibe o resultado na tela.
